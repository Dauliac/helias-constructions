## Rédaction
Pour la rédaction des différentes zones de textes, il faut écrire dans les guillemets `""`

Pour obtenir du gras: entourer le texte de **deux astériques**: `**En gras**`.

Pour l'*italique* ça sera d'une seule. `*En italique*`
